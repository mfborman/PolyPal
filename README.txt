Format for GIT commit comments:

  "Short 10 or less word description"

  In file "filename"
    - "Update1"
    - "Update2"
    - ...

How to edit the .gitignore file
  1. Navigate to the project home directory
  2. Run the command: mv .gitignore gitignore.txt
  3. Edit the file with: vim gitignore.txt
  4. After saving the file, run: mv gitignore.txt .gitignore
